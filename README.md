# lobe_analysis


## before using, ensure you have pipx and poetry:


install pipx
```
python3 -m pip install --user pipx
python3 -m pipx ensurepath
```

install poetry
```
pipx install poetry
```


1. Clone repository


2. If using poetry/pipx:

```
poetry install
poetry shell
```

3. otherwise:

```
python3.11 -m venv venv
source venv/bin/activate
pip install .
```
